.. _is-md5:

isMd5
=====

``isMd5`` checks that the string is a ``md5`` format, i.r. a hexadecimal string of 32 length.

.. code-block:: php

   <?php
   $hash    = hash('md5', 'atoum');
   $notHash = 'atoum';

   $this
       ->hash($hash)
           ->isMd5()       // passes
       ->hash($notHash)
           ->isMd5()       // fails
   ;

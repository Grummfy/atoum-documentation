.. _is-md5:

isMd5
=====

``isMd5`` vérifie que la chaîne de caractère est du format ``md5``, par exemple une chaîne de caractère d'une longueur de 32.

.. code-block:: php

   <?php
   $hash    = hash('md5', 'atoum');
   $notHash = 'atoum';

   $this
       ->hash($hash)
           ->isMd5()       // passe
       ->hash($notHash)
           ->isMd5()       // échoue
   ;
